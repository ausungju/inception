# Description
이 프로젝트는 Docker를 활용하여 시스템관리를 학습한다.

여러가지 프로그램(NGINX, MariaDB, fpm)를 설정하고 docker compose를 활용하여 하나의 통합적인 서비스를 구성한다.

docker를 사용하여 HOST 시스템과 서비스가 작동하는 시스템을 분리, 재현성을 보장한다.

# Instructions
```bash
sudo apt update
sudo apt install docker docker-compose-plugin
```
